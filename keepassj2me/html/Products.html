PhoneID
PhoneID is a framework for secure user authentication using a mobile phone. A J2ME enabled mobile phone would encrypts the user passwords and keys both on the phone and hte host and store those infomration on the phone nad hte host.   The phone would send passwords, carries out challenge / response, encryption / decryption with a host computer, automatically through Bluetooth.

KeePass J2ME
KeePass for J2ME is a J2ME port of KeePass Password Safe, a free, open source, light-weight and easy-to-use password manager. You can store passwords in a highly-encrypted database on a mobile phone, and view them on the go.
